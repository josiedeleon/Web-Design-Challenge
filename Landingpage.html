WeatherPy



The purpose of this project was to analyze how weather changes as you get closer to the equator. First we pulled data from the OpenWeatherMap API for over 500 cities. Then we used Matplotlib to plot weather vs latitude for various factors. Factors considered were temperature, cloudiness, wind speed, and humidity. This site provides the source data, visualizations, and explanations of any trends and correslations identified.
